🔹 **Shell Scripting & Linux Commands**  
- Basic shell scripting  
- File handling, loops, and automation  
- Process management (`ps`, `kill`, `bg`, `fg`)  
- Scheduling (`cron`, `at`)

- **Process Creation:** `fork()`, `exec()`, `wait()`
